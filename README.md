# Plateforme de Traitement de Signal - Scilab

## Description
Interface interactive pour 5 cas pratiques de traitement du signal :
1. Nettoyage audio de podcast
2. Amélioration d'image médicale
3. Détection R-peaks ECG
4. Estimation distance radar
5. Simulation liaison radio

## Prérequis
- Scilab 6.0 ou supérieur
- Windows/Linux/MacOS

## Installation
1. Cloner le dépôt
2. Ouvrir Scilab
3. Exécuter : `exec('src/interface_principale.sce')`
